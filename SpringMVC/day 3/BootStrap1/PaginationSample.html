Bootstrap Pagination

Pagination is used to sort the web pages of your website in an organized manner. It becomes very necessary if your website has a lot of web pages. Following is a list of classes that Bootstrap provides to handle pagination.

Class Description
.pagination Add this class to get the pagination on your page.
.active,
.disable		 you can customize links by using .disabled for unclickable links and .active to indicate the current page.
.pagination-lg,
.pagination-sm		 use these classes to get different size items.

A basic pagination example:

Add the .pagination class to an <ul> element to create a basic pagination.
